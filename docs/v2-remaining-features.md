## ❌ 미구현 항목

### 🎧 2. 플레이어 화면 UI/UX 개선

#### 가. 몰입형 문장 보기
- ❌ **현재 문장 중앙 집중**: `currentSentence`를 화면 중앙에 큰 폰트로 표시
- ❌ **컨텍스트 제공**: 위/아래에 이전/다음 문장을 흐리게(semi-transparent) 또는 더 작은 폰트로 배치
- ❌ **자동 스크롤**: `currentPositionMs`가 변경될 때마다 현재 문장이 부드럽게 중앙으로 스크롤되도록 애니메이션 적용

**현재 상태**: 모든 문장이 스크롤되는 목록 형태

#### 다. 컨트롤 UI 재배치
- ❌ **주요 버튼**: '재생/일시정지' 버튼을 가장 크고 중앙에 배치
- ❌ **보조 버튼**: '-10초', '+10초' 버튼을 그 옆에 배치
- ❌ **기타 기능**: '재생 속도'나 'A-B 반복' 같은 기능은 작은 아이콘(예: 1.5x, A↔B)으로 탐색 바 근처나 상단 헤더에 배치

**현재 상태**: 재생/일시정지, 탐색 바, -10초/+10초가 나열되어 있음

---

### 📚 3. 에피소드 목록 (홈 화면) 개선

#### 가. 카드(Card) 디자인 및 진행률 표시
- ❌ **카드 디자인**: 각 에피소드를 '카드' 형태로 디자인
- ❌ **썸네일 이미지**: `episodes.thumbnail_path`를 활용한 썸네일 이미지 포함
- ❌ **학습 진행률**: `episode_progress` 테이블의 `current_position_ms`와 `episodes.duration_ms`를 이용해 **학습 진행률(Progress Bar)**을 카드에 작게 표시

**현재 상태**: ✅ 필터링 UI는 이미 구현됨

---

### ⚙️ 설정 페이지 기능

#### 1. 프로필 (Profile)
- ❌ **닉네임 수정**: 탭하면 닉네임을 수정할 수 있는 새 화면(또는 모달)으로 이동. '저장' 시 `app_users` 테이블 업데이트
- ❌ **로그아웃**: 탭하면 `supabase.auth.signOut()` 호출하고 사용자를 로그인 페이지로 이동

**현재 상태**: ✅ UI 구조만 있음 (기능 없음)

#### 2. 학습 설정 (Learning Preferences)
- ❌ **선호 난이도**: 탭하면 1~5까지 선택할 수 있는 옵션(Bottom Sheet 또는 새 화면)이 나타나고, `difficulty_preference`를 업데이트
- ❌ **일일 학습 목표**: 탭하면 시간(분)을 선택할 수 있는 휠 피커(Wheel Picker) 등이 나타나고, `daily_goal_minutes`를 업데이트
- ❌ **SRS 복습 주기**: 표시만 하거나, Phase 2에서 수정 가능하게

**현재 상태**: ✅ UI만 있음 (기능 없음)

#### 3. 알림 (Notifications)
- ❌ **학습 알림 받기**: 토글 스위치(Toggle Switch)를 배치하고, 토글을 켜고 끌 때마다 `notification_enabled` (boolean) 값을 즉시 업데이트

**현재 상태**: ✅ 아이콘만 있음 (실제 토글 기능 없음)

---

### ✨ 5. 전반적인 폴리시 (Polishing)

#### 스켈레톤 로더 (Skeleton Loaders)
- ❌ 에피소드 목록을 불러올 때, '로딩 중...' 텍스트 대신 회색의 스켈레톤 UI를 보여주기

**현재 상태**: ActivityIndicator만 사용

#### 페이지 전환 (Page Transitions)
- ❌ 페이지 이동 시(예: 목록 → 플레이어) 부드러운 슬라이드 효과 추가

**현재 상태**: 기본 전환만 사용

#### 상태 피드백 (State Feedback)
- ❌ 버튼을 탭했을 때(예: 단어 저장), 버튼이 잠시 로딩 상태(spinner)로 바뀌거나 성공 시 체크 표시 애니메이션을 보여주기

**현재 상태**: 일부만 구현됨 (단어 저장 시 피드백 없음)

---

### 🗂️ 단어장 페이지 추가 기능

#### 검색 기능
- ❌ **실제 검색 기능**: 검색 바 UI만 있고 실제 검색 동작 없음
- ❌ **Debounce**: 검색 입력 시 300ms debounce 적용

**현재 상태**: ✅ UI만 있음

#### 레벨별 탭
- ❌ **레벨별 그룹화**: `user_word_progress.level` 별로 단어를 그룹화하여 보여주기 (예: "Level 0: 20개", "Level 1: 15개")

**현재 상태**: ✅ 탭 UI만 있음 (실제 그룹화 없음)
